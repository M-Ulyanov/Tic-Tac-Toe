## Tic-Tac-Toe
Всем знакомая игра "Крестики-Нолики"

### Возможности
- Выбор сложности игры
- Выбор размера поля 
- Общая статистика всех сыгранных игр
- Динамическая инфографика

### Кроссбраузерность
- Все современные браузеры
- IE8+
